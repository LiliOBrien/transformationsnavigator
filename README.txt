
# Website-Starter (DE)

Diese Dateien kannst du direkt bei GitHub hochladen und mit Vercel deployen.

## Schnelle Schritte
1. ZIP entpacken.
2. Auf GitHub ein neues Repository anlegen (z. B. `navigator-website`).
3. `index.html`, `style.css` und den Ordner `assets` hochladen.
4. Auf Vercel ein neues Projekt aus dem Repository erstellen → Deploy.

## Wo anpassen?
- **Name & E-Mail**: in `index.html` im Abschnitt „Kontakt & rechtliche Hinweise“.
- **Navigator-URL**: `index.html` → im iFrame `src="about:blank"` durch deine echte Navigator-URL ersetzen.
- **Texte/Farben**: `index.html` (Texte) und `style.css` (Farben/Schrift).

Viel Erfolg!
